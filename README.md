# Personal Expense Tracker

A simple command-line expense tracking application built with Python and Pandas.

## Features
- Add expenses with category, amount, and description
- Automatic date tracking
- Persistent storage using CSV
- View all expenses in a formatted table

## Technologies Used
- Python 3
- Pandas for data manipulation
- CSV for data persistence

## Installation
On bash:
 - pip install pandas

## Usage
bash:
  - python3 expense_tracker.py

Follow the prompts to add expenses.
